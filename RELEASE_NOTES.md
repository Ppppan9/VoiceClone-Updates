# Release 1.0.7

## ✨ 更新说明
- 🎯 顶部“配音结果”状态现在只显示 `成功`、`失败`、`无`，结果判断更直接。
- ✂️ 文字配音区域移除了音频预览，界面更简洁，操作更聚焦。
- 📁 新生成的配音文件会保存到当前 AE 工程目录下的 `ElevenLabs` 文件夹中。
- 🗂️ 导入 AE 后，音频素材也会自动归到项目面板里的 `ElevenLabs` 文件夹。
- 🔄 声音克隆区域的“刷新声音库”按钮已精简为“刷新”。
