# Release 1.0.5

## ✨ 更新说明

- 🔍 文字配音现在支持按声音名、来源和 `voice_id` 进行关键词模糊搜索，找声音更快。
- 🗑️ 克隆声音区域新增“选中”和“删除选中”能力，清理声音库更方便。
- 🎛️ 已克隆声音列表改成更精简的滚动预览，只保留声音名称和来源信息。
- 🎨 面板布局继续优化，主功能更聚焦，操作路径更直接。
